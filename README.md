# 项目简介

本项目是一个基于Vue的仿美团外卖App的练习项目。

使用Vue+NodeJS+Express+MongoDB开发 。

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
